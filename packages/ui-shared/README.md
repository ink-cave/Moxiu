# @moxiu/ui-shared

Shared UI utilities and components.

Install

```
npm install @moxiu/ui-shared
```

Build / Develop

- Build: `npm run build`
- Dev (watch): `npm run dev`
- Test: `npm run test`
- Lint: `npm run lint`

Usage

Import from the package:

```ts
import { foo } from '@moxiu/ui-shared'
```
